# Clear Sky

Clear Sky is more than just a game project; it is the culmination of a long-term creative journey. At its heart is a character I created years ago, now finally brought to life through a blend of code and digital artistry.

## 🎨 A One-Man Studio
This project allowed me to manage the full creative and technical pipeline:
* The Vision: Bringing a legacy character into a playable 2D environment.
* The Craft: Hand-drawn 2D art and animations designed to capture a specific mood and flow.
* The Sound: An original soundtrack and sound effects composed to immerse the player in the world of Clear Sky.
* The Logic: Using Unity and C# to build a responsive, modular framework that feels as fluid as the art looks.

## 🛠 Built With
* Engine: Unity 6.3
* Language: C#
* Art & Animation: Photoshop
* Audio: Original composition

## ✨ Why This Matters
This project is about the challenge of mastering new tools—whether it’s refining a game loop or mixing a music track—to ensure that the final result is a cohesive, polished experience.
